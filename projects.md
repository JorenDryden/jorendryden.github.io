#Role: Project Lead
#About: TurtleShell is an locally hosted encryption manager for documents, passwords, and other data. The focus of the project was to implement indsustry standard encrpytion methods in conjunction with a friendly and asthetically pleasing user interface. TurtleShell was written mainly in Java and takes use of Java's crpyto library and used JavaFX for the GUI.
